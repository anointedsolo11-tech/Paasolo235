# My Todo App ✅

A simple, offline-capable Todo App with PWA support.

## Features
- ✅ Add and delete tasks
- 🌙 Dark mode toggle
- 💾 Local storage (tasks persist after refresh)
- 📱 Install as app on phone/computer
- 📡 Works offline

## Technologies
- HTML5, CSS3, JavaScript
- PWA (Service Worker, Manifest)
- LocalStorage API

## How to use
1. Type a task and click "Add" or press Enter
2. Click "X" to delete a task
3. Click "Dark Mode" for night theme
4. Click "Install" to install as app
